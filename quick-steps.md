1. **Withdraw STG from Coinbase**:

2. **Set Up Wallet** (If New):

3. **Swap on Portal**:
   - Go: [stargate.finance](https://stargate.finance/) (bookmark it!).
   - Connect wallet > Select STG > Enter amount > Swap to ZRO.
   - Approve (once) + Confirm. Gas: $10-50 (cheaper on Base if supported).

4. **Deposit ZRO Back?**:
   - Coinbase: Assets > ZRO > Receive > Send from wallet.

Done! Track ZRO on [LayerZero](https://layerzero.network). If stuck: [Coinbase Help](https://help.coinbase.com/en).

Pro Tip: Use Base chain for lower fees if the portal allows.
